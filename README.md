## Coding Platform Features
- A competitive coding portal where people can read, registred users can comment and add new questions can discuss questions solutions and collaborate. 
- Login/Register to the Platform.
- Registred User can Post and Comment on Posts
- New Users can read Posts and comments but need to login before posting or commenting
- Recoil for State Management of Logged In User.
- Featured Posts are showcased on Top.
- Tailwind of Styling
- Verel for deployment
